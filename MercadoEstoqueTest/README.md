# Mercado Estoque - Projeto de Testes

Este projeto contém os testes unitários desenvolvidos para o sistema
**Mercado Estoque**, como parte da Etapa 7 da disciplina.

## Objetivo
Validar regras de negócio isoladas da aplicação, utilizando testes
unitários com JUnit, sem dependência de banco de dados.

## Tecnologias utilizadas
- Java
- Maven
- JUnit 5
- NetBeans

## Testes implementados
- Teste unitário da classe `EstoqueService`
- Validação de regras simples de cálculo/negócio

## Observações
Os testes foram separados do projeto principal para garantir
independência, melhor organização e aderência aos princípios SOLID.
